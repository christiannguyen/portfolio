flexbox is pretty cool.

3/11/2017

Another powerful way to create CSS grids and layouts is by using flexbox. I haven't gotten good at flexbox yet but it's something that I defintely want to know really well. Some resources that I have found to be pretty helpful and fun are:

  • flexbox defense
  • flexbox froggy
  • medium guide #1
  • medium guide #2
  • css-tricks

Flexbox works on most browsers except for IE. It's also somewhat buggy on certain browsers such as Firefox but it's not a huge issue. All of these guides and tutorials are really useful if you want to learn flexbox. The visual aid helps a lot with understand how the container or items are displayed. I'll eventually get to learning flexbox and master it by creating some basic website that utilizes only flexbox. Some ideas come in to mind such as a card layout or maybe even a similiar game to flexboxfroggy and flexboxdefense. These tutorials basically act as a puzzle and makes learning much more interesting.

I'm hoping to get back and update this blog post on what I learn about flexbox and when I actually create something using it. Until then this will suffice. Pretty short and sweet I guess. If you're learning or plan on learning to use flexbox check out the links I posted above!
