Projects

Find Me a Recipe!

HTML5, CSS3, Javascript, Jquery, Bootstrap

Used an online recipe API to generate a random recipe for the user to decide on. The API would supply the recipe ID, ingredients, link to the recipe, photo, and name. I grabbed this information using request calls, specifically AJAX.

This project introduced me to APIs and helped me understand how request calls worked. APIs are also really fun to play around with and I look forward to finding out interesting ones to work with!
